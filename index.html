História dos Computadores

(Voltar)

O surgimento dos computadores foram divididos em gerações:

1ª Geração

Harvard Mark I (em 1944), que era basicamente uma máquina muito grande para cálculos complexos, à base de válvulas e sistema eletromecânico.

Em 1946 surguiu o Colossus, para cálculos, criptografias, criptoanálises que pelo lado alemão foi muito utilizado durante a Guerra. Também em 1946, veio o Eniac, do lado dos Estados Unidos. Era mais automático do que eletromecânico.

2ª Geração

Considerado de 1959 a 1965, foi a evolução das válculas para os transistores, que fez com que os computadores diminuíssem drasticmente de tamanho, calro que ainda eram enormes comparados aos computadores atuais.

3ª Geração

De 1965 a 1970, tem os circuitos integrados, os chips, que fizeram diminuir ainda mais os computadores mas aumentando sua eficácia.

4ª Geração

A partir de 1971 até os dias atuais, passaram de circuitos intregardos para microprocessadores. Marcada pelo Personal Computers(PC), como o Apple I, Macintosh, entre outros...

5ª Geração

Muitos dizem que já estamos na 5ª Geração, outros não, entretanto dizem que são os computadores quânticos, as Inteligências Artificiais, criptomoedas, metaverso e muitas outras tecnologias que podem surgir.

O que são Clients?

(Voltar)

Clientes são os computadores que fornecem a interface do usuário e de onde os usuários pode acessar os dados e aplicativos que foram solicitados a partir do servidor . É o dispositivo que está consumindo alguma informação na internet.

O que são Serves?

(Voltar)

O servidor é o computador ou software central que forma uma arquitetura conhecida como “cliente – servidor” com outros dispositivos, como computadores, tablets e smartphones. O servidor, portanto, centraliza todas as informações e demais dados dentro de uma rede.

O que é o HTML

(Voltar)

HTML (Linguagem de Marcação de Hipertexto - HyperText Markup Language) é o código que você usa para estruturar uma página web e seu conteúdo. Por exemplo, o conteúdo pode ser estruturado em parágrafos, em uma lista com marcadores ou usando imagens e tabelas. Como o título sugere, este artigo fornecerá uma compreensão básica do HTML e suas funções. O HTML não é uma linguagem de programação.

Listas ordenadas e não ordenadas

(Voltar)

Lista Ordenada define uma lista de itens em que a ordem dos itens é importante. Também é chamada de lista de números, enquanto a Lista Não Ordenada define uma lista de itens em que a ordem dos itens não define. Também é conhecido como lista com marcadores.
